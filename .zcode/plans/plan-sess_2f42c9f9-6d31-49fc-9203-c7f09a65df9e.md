## TrustNet — Bold Redesign + Scanner Demo + Full SEO/GEO

### What this is for (kept front-of-mind)
TrustNet is a **digital-trust / anti-fraud platform for India** — "सुरक्षित इंटरनेट, सुरक्षित भारत।" Its job: help everyday citizens **verify before they click, share, or pay** — phishing sites, scam SMS (KYC/lottery/delivery), fake news, deepfakes. Bilingual EN/हिन्दी. Every change below serves that mission.

The Express server (auth routes) is left untouched.

---

### A. New theme / color system (bold redesign) — `index.css` + tokens
- **New palette with semantic meaning**: deep trust-navy primary, **emerald "verified safe" accent**, red/rose "danger", amber "caution". (Green-for-safe directly reinforces the "is this safe?" mental model.)
- Keep the premium **skeuomorphic/glass tactile feel** (you invested in it) but refine it: softer shadows, refined gradients, subtle mesh-gradient background.
- **Class-based dark mode** (add `@custom-variant dark`) + convert all `.skeuo-*` `@media (prefers-color-scheme)` rules to `.dark`-scoped rules so the manual toggle works.
- Fix all **dead Tailwind classes** found in audit: `text-blue-405`, `w-4.5/h-4.5`, `w-5.5/h-5.5`, `text-slate-550`, `border-slate-350`, `bg-slate-850`, `focus:ring-3`, etc.
- Keep **Outfit + Mukta** (Mukta renders Devanagari — essential for Hindi).
- Add `prefers-reduced-motion` handling.

### B. Manual dark/light toggle — new `theme.tsx` + `components/ThemeToggle.tsx`
- `ThemeProvider` with **localStorage persistence + system-preference default**, toggling `.dark` on `<html>`.
- Reusable Sun/Moon toggle in nav, login, signup.

### C. ⭐ Interactive Scanner demo (the missing core) — new `components/Scanner.tsx` + `utils/scannerLogic.ts`
- Front-end-only, clearly labeled **"Demo"**. Three branches: **Website / SMS / News** (Image mentioned as "coming soon").
- Pure heuristic detection (`scannerLogic.ts`) returns `{ score, level: safe|caution|danger, signals[] }`:
  - **URL**: lookalike domains (e.g. `sbi` → `sbi-secure-portal`), excess hyphens, IP-as-host, suspicious TLDs, "recently registered" keyword cues.
  - **SMS**: KYC-blocked, lottery/prize, delivery fee, OTP/UPI/click-link requests, urgency language.
  - **News**: match against a small in-file list of known circulating fakes + sensationalist cues.
- Verdict card reuses the SVG risk gauge, with a clear Safe/Caution/Danger badge + signal breakdown. Wired into the **EN/HI translation system**.
- Placed right after the Hero — strongest "show, don't tell."
